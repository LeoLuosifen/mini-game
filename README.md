# 在线小游戏平台

欢迎来到在线小游戏平台，这里汇集了多种类型的休闲小游戏，为玩家提供丰富的娱乐体验。

## 平台游戏

### 凡人修仙传

这是一个修仙主题的放置类游戏，玩家可以通过修炼、历练、打工等方式提升修为，突破境界，最终成为修仙界的顶尖存在。

**游戏特色**：
- **灵根系统**：多种灵根类型，每种灵根有不同的修炼速度和境界上限
- **境界系统**：从凡人到大帝的完整修仙境界体系
- **洞府系统**：升级洞府提升修炼效率和特殊能力
- **宗门系统**：选择不同的宗门，获得不同的修炼加成
- **历练系统**：外出历练获得资源和机缘
- **打工系统**：完成任务获得灵石
- **坊市系统**：购买丹药和道具
- **渡劫系统**：突破大境界时需要渡劫
- **转生系统**：兵解转生获得新的灵根和机会

**游戏玩法**：
1. **初始化**：输入名字，随机获得灵根，选择宗门
2. **修炼**：自动修炼或手动修炼提升修为
3. **突破**：修为达到要求后突破境界
4. **历练**：外出历练获得资源和机缘
5. **打工**：完成任务获得灵石
6. **坊市**：购买丹药和道具
7. **洞府**：升级洞府提升修炼效率
8. **转生**：当遇到瓶颈时，兵解转生获得新的灵根和机会

### 其他游戏

- **消消乐**：经典的三消游戏，通过消除相同颜色的方块获得分数
- **贪吃蛇**：控制蛇头吃食物，不断增长身体，避免撞到墙壁和自身
- **俄罗斯方块**：经典的方块下落游戏，通过旋转和移动方块消除行
- **五子棋**：传统的黑白棋游戏，先在一条直线上连成五个棋子的一方获胜
- **2048**：通过合并相同数字的方块，获得2048方块

## 运行说明

### 本地运行

**前提条件:** Node.js

1. 安装依赖：
   `npm install`
2. 运行应用：
   `npm run dev`

### 构建部署

1. 构建应用：
   `npm run build`
2. 部署构建后的文件到你的服务器

## 平台特色

- **多样化游戏**：涵盖休闲、策略、益智等多种类型
- **无需下载**：直接在浏览器中运行，无需安装
- **跨平台**：支持PC、手机、平板等多种设备
- **数据保存**：支持本地存储，保存游戏进度
- **定期更新**：持续添加新游戏和功能

## 平台集成

该平台可作为独立网站运行，也可以集成到其他网站中。每个游戏都可以通过iframe单独嵌入，或者作为平台的一部分整体访问。
